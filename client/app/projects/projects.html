  • Loudspkr

    Project Leader

    A website that places a virtual localized message board at every venue in the US based on the Foursquare and Firebase APIs. Allows users to meet new people, organize events, and broadcast messages in the current location through messages. Created for HackNY Spring 2014.

  • Facial Recognition Sentry

    Project Leader, Computer Vision

    This is a hack made for Hack Cooper 2014 that can search for faces with a webcam using a Haar Cascade OpenCV and then turn toward the face with a Nerf gun and shoot using an Arduino.

  • Automatic Blood Pressure Monitor

    Group Leader, Software Developer

    This is a project made for Freshman Engineering class that measures systolic and diastolic pressures using a TI DSP receiving signal from a cuff pump and a contact mic.

  • Web Explorer

  • Drop

    Project Leader, Frontend Developer

    This is a web-based application project made for the Stuyvesant Software Development class that allows the user to drop a note in his/her current HTML5 geolocation. Other users can read notes geographically close to themselves and see where notes exist in the world. There is also extensive javascript work for the frontend that provides an animated interface.

  • Minecarft

    Mid-term project for AP Computer Science. A Minecraft-esque landscape-generation (using Perlin noise) program with a first-person camera (in OpenGL) for exploring the generated landscape.
